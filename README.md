# Portfolio professionnel - Mayssae Bentayeb 
Ce dépot contient mon **portfolio professionnel multi-page** réalisé avec **Jekyll** et **Markdown**, et déployé automatiquement via **GitHub Pages**.
---
## Technologies utiliséees
- HTML
- Scss
- Jekyll
- Git & GitHub
- GitHub Page
---
## Structure du site
Le portfolio est composé de plusieurs pages distinctes:
- **Accueil** : présentation rapide de profil
- **À propos** : parcours, compétences et information personnelles
- **Projects** : sélection de projects réalisée avec descriptions et liens GitHub
- **Contact** : informationd de contact et liens professionnels
